The course provides the mathematical and numerical foundations for extracting information from large datasets. It explores advanced algebraic tools designed to manipulate and analyze high-dimensional data, emphasizing the trade-offs between computational cost, stability, and accuracy.

Key topics covered include:
- **Matrix Factorizations:** QR factorization (Givens, Householder), SVD, and Moore-Penrose pseudo-inverse.
- **Large-scale Systems:** Iterative solvers, preconditioning, and eigenvalue computations for sparse matrices.
- **Data Approximation:** Least-squares, interpolation, and randomized SVD (Johnson-Lindenstrauss theorem).
- **Dimensionality Reduction:** Theoretical properties and applications of Principal Component Analysis (PCA).
- **Graph and Data Mining:** Spectral Clustering and iterative methods for ranking.
